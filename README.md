# dotfiles-win

dotfiles to be used on Windows

Installed programs:
- git for Windows
- VsVim for Visual Studio

### Install

Clone this repository into `~/.dotfiles`.

Move all files, except the `.git` directory, from `~/.dotfiles` to `~`.

Create `.bashrc` and `.vimrc`, which should source `.my.sh` and `.my.vim`, respectively. Examples are given at the top of `.my.sh` and `.my.vim`.

### Develop

To edit this repository, open `git bash` to `~`, and use the command `dgit` in place of `git`.
